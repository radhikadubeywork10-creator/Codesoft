# Calculator

# Description
This is a simple Calculator project developed using Python. It performs basic arithmetic operations such as addition, subtraction, multiplication, and division.

## Features
* Addition
* Subtraction
* Multiplication
* Division
* User-friendly console interface
* Option to perform calculations multiple times

## Technologies Used
 * Python

## How to Run
1. Open the Python file.
2. Run the program.
3. Enter two numbers.
4. Select the desired operation.
5. View the result.

## Learning Outcome
This project helped me understand Python basics such as variables, user input, loops, conditional statements, and arithmetic operations.
