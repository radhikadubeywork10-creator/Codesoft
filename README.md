# Calculator

* Subtraction
* Multiplication
* Division
* User-friendly console interface
* Option to perform calculations multiple times

1. Open the Python file.
2. Run the program.

5. View the result.

# Password Generator

## Description
This is a Python-based Password Generator that creates secure and random passwords using letters, numbers, and special characters. The program also checks the strength of the generated password.

## Features
* Generates random passwords
* Uses letters, numbers, and symbols
* Password strength checking
* User-friendly console interface
* Option to generate multiple passwords

## Technologies Used
* Python

## How to Run
1. Open the Python file in a Python environment.
2. Run the program.
3. Enter the desired password length.
4. View the generated password and its strength.
5. Choose whether to generate another password.

## Learning Outcome
This project helped me understand:
* Python modules (`random` and `string`)
* Loops (`for` and `while`)
* Conditional statements (`if`, `elif`, `else`)
* User input handling
* String manipulation

This project was developed as part of the CodeSoft Python Programming Internship.
