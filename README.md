## Rock Paper Scissors Game

### Description
This is a simple Rock Paper Scissors game developed using Python. The user plays against the computer by choosing Rock, Paper, or Scissors. The computer makes a random choice, and the winner is determined based on the game rules. The game also provides an option to continue playing multiple rounds.

### Features
* User vs Computer gameplay
* Random computer choice using Python's random module
* Win, Lose, or Tie result display
* Input validation for incorrect choices
* Option to play multiple rounds
* Simple and user-friendly interface

### Technologies Used
* Python
* Random Module

### How to Run
1. Install Python on your system.
2. Download or clone the project.
3. Open a terminal in the project directory.
4. Run the following command:

```bash
python rock_paper_scissors.py
```

### Game Rules
* Rock beats Scissors
* Scissors beats Paper
* Paper beats Rock
* If both choices are the same, the result is a Tie

### Learning Outcomes
This project helped in understanding:
* Conditional Statements (if-elif-else)
* Loops
* User Input Handling
* Random Number Generation
* Basic Game Logic
